fn.attr()

Basic setter/getter for Context properties.

 $("#canvas").attr('globalAlpha', 0.5);
 $("#canvas").attr('globalAlpha');

You can also set multiple attributes at once:

 $("#canvas").attr({
   globalAlpha: 0.5,
   globalCompositeOperation: 'source-over'
 })

You don't often need to use this, as there are aliases available for most attributes already. For instance, those .attr('globalAlpha') calls above are the same as:

 $("#canvas").globalAlpha(0.5);
 $("#canvas").globalAlpha();

  C3.fn.attr = function(key, value) {
    if (typeof key === 'object') {
      for (var attr in key) {
        if (key.hasOwnProperty(attr)) {
          this.setAttr(attr, key[attr]);
        }
      }
    }
    else if (typeof value === 'undefined') {
      return this.getAttr(key);
    } else {
      this.setAttr(key, value);
      return this;
    }
  };

fn.run()

Performs a function call method to the Canvas Context, passing the arguments array args.

 $c("#canvas").run("stroke")

 // Same as canvas.getContext("2d").stroke();

You can pass arguments as an array.

$c("#canvas").run("moveTo", [10, 40]);

// Same as canvas.getContext("2d").moveTo(10, 40);

You can get the result via .result.

 $c("#canvas").run("getImageData", [0,1,2,3]).result;

You don't often need to use this, as there are aliases available for most attributes already. For instance, the .run("moveTo", [10, 40]) call above is the same as:

 $("#canvas").moveTo(10, 40);

Internally, all canvas context calls pass through here.

 
  C3.fn.run = function(method, args) {
    this.result = this.cx[method].apply(this.cx, args);
    return this;
  };

How to draw

You will usually need to use .style(), some drawing functions, and .paint() in that order:

$c("#canvas")
  .style({ fillColor: 'black' })
  .rect(x,y,w,h)
  .paint()

The .style() sets the ink styles. The drawing functions (like .rect(), .line(), ...) will draw paths, while the .paint() call will paint by filling/stroking the styles set earlier.

    var self = this;
    this.beginPath();

      if (list.hasOwnProperty(index)) {

Command compilation



Youc an compile Canvas calls into an anonymous function using .compile()
and .compileEnd(). Useful for 'caching' things, or maybe using c3 in a
web worker.



var fn = $c("#canvas")
  .compile()
  .style({ fillStyle: 'red' })
  .line([[0,0], [1,10])
  .paint()
  .compileEnd()




The result is a flat, anonymous function. It will have no closure
dependencies, so it's perfectly useable as a string, as well.



console.log(fn.toString());

/* Result:
   function(cx) {
     cx.fillStyle = 'red';
     cx.moveTo(0,0);
     cx.lineTo(1,10);
     cx.fill();
   }
*/




The resulting function takes a Canvas context.



fn(canvas.getContext('2d'));

             		                            
  C3.fn.compile = function() {
    this.commands = [];
    $.extend(this, C3.Compiler);
    return this;
  };

  C3.Compiler = {
             		           
                               
                            
                 ¶               
               
Temporary getAttr() replacement: does nothing.
             		                            
    getAttr: function(attr) {
      return undefined;
    },
             		           
                               
                            
                 ¶               
               
Temporary setAttr() replacement to cache the attribute set commands.
             		                            
    setAttr: function(attr, value) {
      this.commands.push("cx." + attr + " = " + JSON.stringify(value) + ";");
      return this;
    },
             		           
                               
                            
                 ¶               
               
Temporary run() replacement to cache the method calls.
             		                            
    run: function(method, args) {
      function toArgs(args) {
        var str = JSON.stringify(Array.prototype.slice.call(args, 0));
        return str.substr(1, str.length - 2);
      };

      this.commands.push("cx." + method + "(" + toArgs(args) + ");");
      return this;
    },
             		           
                               
                            
                 ¶               
               
Terminate compilation by returning commands to defaults, and baking the
commands into an anonymous function.
             		                            
    compileEnd: function() {
      var commands = this.commands;

      delete(this.commands);
      delete(this.getAttr);
      delete(this.setAttr);
      delete(this.run);
      delete(this.compileEnd);

      return new Function("cx", commands.join("\n"));
    }
  };
