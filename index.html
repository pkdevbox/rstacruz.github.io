I am an open-source contributor and I like making things that help people with their work. I mainly work with Ruby and JavaScript projects, both self-started projects and community projects. I am the author of:

  • Backbone Patterns is a guide that documents common usage patterns when making HTML5 apps with Backbone.js.

  • jQuery Transit brings the power of fast CSS3 transitions to jQuery.

  • Js2coffee is a JavaScript compiler that outputs CoffeeScript code, helping people learn CoffeeScript and aiding in migrating big projects to CoffeeScript.

  • Ubuntu Dust is a UI theme for the Ubuntu OS that has been included as a default theme since Intrepid.

  • Sinatra AssetPack is a lightweight asset packager for Sinatra that compresses CSS and JavaScript for fast web applications.

  • Mina is a fast deployment tool to get web applications running. This was co-authored with my colleague Michael Galero at Nadarei.

My open source projects often got featured in The Changelog podcast & blog and Hacker News, among other venues. I also have a few other smaller projects under my belt.
