Elements reference

Elements reference

These are the common available elements:

  • Block elements

    • p (paragraph)
    • h1, h2, h3, h4, h5, h6 (heading)
    • ul, li (list)
    • div

  • Inline elements

    • a (link)
    • strong (bold)
    • em (emphasis)
    • span

Lists and list items

Lists and list items

  • UL is for unordered lists.
  • LI is for list elements inside that list.

HTML

<ul>
  <li>
    <strong>UL</strong>
    is for unordered lists.
  </li>
  <li>
    <strong>LI</strong>
    is for list elements inside that list.
  </li>
</ul>
</ul>

Font styling 2

Font styling 2

Bold. Underline. Italic.

HTML

<span class='bold'>Bold.</span>
<span class='underline'>Underline.</span>
<span class='italic'>Italic.</span>

CSS

.bold {
  font-weight: bold; /* or 'normal' */
}
.italic {
  font-style: italic; /* or 'normal' */
}
.underline {
  text-decoration: underline; /* or 'none' */
}
